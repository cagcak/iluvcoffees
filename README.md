## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Pre requentings

### Install docker-compose

Check first the latest release version at <a href="https://github.com/docker/compose/releases">release page</a>

> as default, version 1.29.2 is currently being used

```bash
$ sudo curl -L "https://github.com/docker/compose/releases/download/<LATEST_RELEASE_VERSION/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
$ sudo chmod +x /usr/local/bin/docker-compose
$ docker-compose --version
```

### Setup postgresql visual tools (optional)

we use pgadmin-4

see the details at <a href="https://www.pgadmin.org/download/pgadmin-4-apt/">installation steps</a>

```bash
#
# Setup the repository
#

# Install the public key for the repository (if not done previously):
sudo curl https://www.pgadmin.org/static/packages_pgadmin_org.pub | sudo apt-key add

# Create the repository configuration file:
sudo sh -c 'echo "deb https://ftp.postgresql.org/pub/pgadmin/pgadmin4/apt/$(lsb_release -cs) pgadmin4 main" > /etc/apt/sources.list.d/pgadmin4.list && apt update'

#
# Install pgAdmin
#

# Install for both desktop and web modes:
sudo apt install pgadmin4

# Install for desktop mode only:
sudo apt install pgadmin4-desktop

# Install for web mode only:
sudo apt install pgadmin4-web

# Configure the webserver, if you installed pgadmin4-web:
sudo /usr/pgadmin4/bin/setup-web.sh
```

lastly see the connectivity settings configuration setup guide <a href="https://www.pgadmin.org/docs/pgadmin4/latest/server_dialog.html">here<a/>

### Setup postgres environment

create a .env file on he project root directory and place the database connection environment settings as below:

```bash
DATABASE_HOST=localhost
DATABASE_PORT=5432
DATABASE_USER=postgres
DATABASE_PASSWORD=pass123
DATABASE_NAME=postgres
API_KEY=*****************
```

### Database migrations

first make a build of the application

```bash
$ yarn start:db
$ yarn build
$ yarn migration:generate src/migrations/coffees/SchemaSync
```

then add the latest generated migration `SchemaSync1657658529129` to typeorm migrations list

```typescript
  ...
  migrations: [CoffeeRefactor1656843398457, SchemaSync1657658529129],
});
```

then run the migrations

```bash
$ yarn build
$ yarn migration:run
```

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# debug mode
$ yarn start:debug

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
